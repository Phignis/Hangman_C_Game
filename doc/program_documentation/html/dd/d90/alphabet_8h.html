Description détaillée

Définit la structure Alphabet, ainsi que le type enum Boolean.

Alphabet et Boolean, ainsi que leur comportement, sont définis. Boolean est un simple enum, Alphabet est un tableau de Boolean, avec sa taille logique et physique

◆ createAlphabet()

Boolean createAlphabet ( Alphabet **  toCreate)

permet de créer un alphabet

déclare un array de Boolean de 26 champs, représentant l'utilisation de chaque lettre de l'alphabet latin. Alloue dynamiquement l'espace pour l'alphabet. ATTENTION!! Utilisez destroyAlphabet pour éviter toutes fuites de mémoire

Paramètres
toCreatepointeur contenant l'adresse de la structure alphabet créée
Renvoie
True si la création est réussie, false sinon

◆ isProposedLetterValid()

Boolean isProposedLetterValid ( const Alphabet  alphabet,
char  submittedChar 
)

Permet de savoir si une lettre proposée est valide, donc non utilisée.

Recherche dans l'alphabet si le caractère est déjà utilisé, donc que le Boolean lié à son index est à True.

Paramètres
alphabetstructure contenant l'état d'utilisation des lettres
submittedCharcaractère proposé par l'utilisateur
Renvoie
2 si le char est '!' 1 si le char est valide (une lettre, non déjà utilisé) 0 si:
  • ce n'est pas une lettre
  • le tableau de l'alphabet n'existe pas (NULL)
  • la lettre n'est pas dans l'alphabet passé en paramètre
  • la lettre est déjà utilisé

◆ printAlphabet()

Affiche pour un jeu du pendu l'alphabet.

N'affiche que les lettres de l'alphabet non encore utilisées durant le jeu, donc dont le Boolean à son index est à False. Affiche un espace à la place de la lettre si cette dernier n'est pas présente

Paramètres
toPrintalphabet à afficher

Permet de mettre à jour les lettres d'un alphabet.

Permet de mettre à True l'état d'utilisation de la lettre dans alphabet, correspondant au charactère soumis

Paramètres
toUpdatealphabet dont il faut update l'état d'utilisation de la lettre propposé à True
submittedCharcaractère dont il faut mettre l'état d'utilisation à True dans l'alphabet
Renvoie
True si la mise a jour a été effectué
False si quelque chose s'est mal passé:
  • le champ array de Alphabet vaut NULL
  • le char n'est pas une lettre
  • la lettre n'est pas dans l'alphabet passé en paramètre

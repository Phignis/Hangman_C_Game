◆ importWords()

Dictionnary * importWords ( FILE *  source)

importe dans un dictionnaire des mots présents dans un fichier source

Le fichier source est attendu comme contenant par ligne un seul mot de 7 charactères max
ATTENTION! il est attendu que le flux passé en paramètre possède des droits de lecture ATTENTION! le flux de données n'est ni ouvert ni fermé dans la fonction. Charge à la fonction appelante de gérer ouverture et fermuture.

Paramètres
sourceflux de données utilisés pour remplir le dictionnaire
Renvoie
NULL si source est NULL ou que la lecture s'est mal passé
adresse du dictionnaire créé sinon. Se référer à destroyDictionnary pour libérer

◆ writeWords()

int writeWords ( FILE *  placeToSave,
Dictionnary  wordsToWrite 
)

permet d'écrire une liste de mots dans un flux

Ecrira les chaînes de charactères présents dans wordsToWrite à partir de l'endroit indiqué par le flux. ATTENTION! donnez un flux possédant des droits d'écritures et sans mode binaire
ATTENTION! le flux de données n'est ni ouvert ni fermé dans la fonction. Charge à la fonction appelante de gérer ouverture et fermuture.
ATTENTION! le nombre de mots écrits rééllement est renvoyé. Si au cours de l'écriture, une erreur survient, il se peut que seulement une partie des mots soient écrits.

Paramètres
placeToSaveflux de données où stocker les mots de wordsToWrite
wordsToWritedictionnaire dont il faut écrire les mots dans le flux donné
Renvoie
-1 si placeToSave ou wordsArray présent dans wordsToWrite vaut NULL
le nombre de mots écrits rééllement sinon

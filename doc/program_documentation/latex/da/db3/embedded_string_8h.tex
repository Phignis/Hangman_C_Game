Définit la structure Embedded\+String, ainsi que le type enum Boolean. 

Embedded\+String et Boolean, ainsi que leur comportement, sont définis. Boolean est un simple enum, Embedded\+String est un tableau d\textquotesingle{}\mbox{\hyperlink{struct_embedded_char}{Embedded\+Char}}.

\doxysubsection{Description détaillée}
\mbox{\hyperlink{struct_alphabet}{Alphabet}} est un wrapper de tableau. 

Outre le tableau contenant des valeurs booléennes, il contient aussi sa taille logique et physique.~\newline
 Le tableau permet de savoir si une lettre, donné par l\textquotesingle{}index, a été utilisé ou non.
